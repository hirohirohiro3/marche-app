# Marche App 管理者操作マニュアル

このマニュアルでは、Marche Appの管理者機能（メニュー管理、売上分析、オプション管理、決済設定、QRコード設定）の操作方法について説明します。

## 1. メニュー管理 (Menu Management)

メニュー画面では、商品の登録、編集、削除、在庫管理を行うことができます。

### 商品の追加
1. 管理画面のサイドメニューから「メニュー管理」を選択します。
2. 画面右上の「商品を追加」ボタンをクリックします。
3. 商品名、価格、カテゴリ、説明を入力します。
4. 商品画像をアップロードします（推奨サイズ: 正方形）。
5. 「保存」ボタンをクリックして商品を登録します。

### 商品の編集・削除
1. メニュー一覧から編集したい商品をクリックします。
2. 内容を修正し「保存」をクリックすると更新されます。
3. 商品を削除する場合は、編集画面内の「削除」ボタンをクリックします（削除前に確認ダイアログが表示されます）。

### 在庫管理
1. 商品編集画面で「在庫管理を行う」にチェックを入れます。
2. 現在の在庫数を入力します。
3. 在庫が0になると、自動的に「売り切れ」状態となり、注文ができなくなります。

## 2. 売上分析 (Sales Analysis)

ダッシュボードおよび分析画面で、売上状況を確認できます。

### ダッシュボード
- **新規注文**: まだ調理・提供されていない注文が表示されます。ステータスを変更して管理します。
- **支払い済み**: 支払いが完了した注文が表示されます。
- **完了済み**: 提供が完了した注文が表示されます。

### 売上データの確認
1. サイドメニューから「売上分析」を選択します。
2. 日別、月別の売上合計、注文数を確認できます。
3. 期間を指定してデータをフィルタリングすることが可能です。

## 3. オプション管理 (Option Management)

商品のトッピングやカスタマイズ（例：大盛り、氷なし、ソース追加など）を管理します。

### オプショングループの作成
1. サイドメニューから「オプション管理」を選択します。
2. 「新規グループ作成」をクリックします。
3. グループ名（例：「ドリンクサイズ」「トッピング」）を入力します。
4. 選択肢（例：「Sサイズ」「Mサイズ」）と、それぞれの追加価格を設定します。
5. 「必須項目にする」か「複数選択可」かを設定します。

### 商品への紐付け
1. 「メニュー管理」で商品を編集します。
2. 「オプション」セクションで、作成したオプショングループを選択して紐付けます。

## 4. 決済設定 (Payment Settings)

Stripeを使用したオンライン決済の設定を行います。

### Stripeアカウントの接続
1. サイドメニューから「決済設定」を選択します。
2. 「Stripeと連携する」ボタンをクリックします。
3. Stripeのオンボーディング画面に遷移するので、指示に従ってアカウント情報や銀行口座情報を入力します。
4. 連携が完了すると、「接続済み」と表示され、クレジットカード決済が利用可能になります。

### レシート設定
1. 決済設定画面の「レシート設定」セクションで、店舗名や住所、電話番号を入力します。
2. これらは電子レシートや領収書に記載されます。

## 5. QRコード設定 (QR Code Settings)

お客様が自身のスマホで注文するためのQRコードを発行します。

### QRコードの生成
1. サイドメニューから「QRコード設定」を選択します。
2. テーブル番号や座席名を入力します（例：「Table 1」「カウンターA」）。
3. 「QRコードを生成」をクリックします。

### QRコードの利用
1. 生成されたQRコードを印刷し、各テーブルに配置します。
2. お客様がQRコードを読み取ると、そのテーブル専用の注文画面が開きます。
3. 注文が入ると、管理画面のダッシュボードにテーブル名付きで通知されます。
